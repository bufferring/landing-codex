## Build

### Development Requirements

- **NodeJS** 22.21.1 or higher.
- **PNPM** 10.28.0 or higher (optional).

### Steps

1. **Clone the repository and enter it:**

   ```bash
   git clone https://github.com/bufferring/landing-codex.git
   cd landing-codex
   ```

2. **Build the project:**

   ```bash
   pnpm install && pnpm build
   ```
